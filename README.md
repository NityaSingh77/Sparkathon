# 🚨 Sirn – Intelligent Retail Network

## 📌 Overview  
Sirn is an intelligent stock management and demand forecasting platform built to help retailers optimize inventory, reduce wastage, and improve decision-making.  

Developed as part of Walmart Sparkathon 2025, Sirn was selected among the Top 11 teams globally, showcasing its real-world impact and scalability potential.

---

## ✨ Features  

- 📊 Demand Forecasting using Machine Learning  
- 📦 Smart Inventory Management  
- 📈 Interactive Dashboard for insights  
- 🔄 Data Processing & Integration  
- ⚡ Scalable Architecture  

---

## 🛠️ Tech Stack  

**Frontend:**  
- React  

**Backend:**  
- Node.js / Python  

**Machine Learning:**  
- TensorFlow  
- Scikit-learn  

**Database:**  
- MongoDB  

**Other Tools:**  
- REST APIs  

---

## 🧠 How It Works  

1. Collect historical sales and inventory data  
2. Clean and preprocess the data  
3. Train ML models for demand prediction  
4. Integrate predictions into dashboard  
5. Provide actionable insights to users

## 🚀 Getting Started  

### Prerequisites  

- Node.js  
- Python  
- npm or yarn  

---

### Installation  

```bash
# Clone the repository
git clone https://github.com/your-username/sirn.git

# Navigate into the project directory
cd sirn

# Install frontend dependencies
npm install

# Start the frontend
npm start
```

---

### Backend / ML Setup  

```bash
# Navigate to backend folder
cd backend

# Install Python dependencies
pip install -r requirements.txt

# Run backend server
python app.py
```
